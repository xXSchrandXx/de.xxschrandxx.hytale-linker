# Setting up Hytale-Linker
[German](#German) | [English](#English)

## English
1. Install and configure [Hytale-API](https://github.com/xXSchrandXx/de.xxschrandxx.hytale-api/blob/main/Setup.md) and all required packages and plugins.
2. Install [Hytale-Linker](https://www.woltlab.com/pluginstore/file/TODO/) via the store code, via the package search or as a file upload.
3. Configure Hytale Linker in the ACP under `Configuration > Options > Hytale > Hytale Linker`. (A server must be activated in the selection for Hytale Linker to work.)
4. Remember the URL for `sendCode`.
5. Remember the URL for `updateNames`.
6. Remember the URL for `getLinked`.
7. Remember the URL for `getUnlinked`.
8. Install [WSC-Linker](https://github.com/xXSchrandXx/WSC-Hytale-Linker/blob/main/Setup.md) on your Hytale server.

## German
1. Installiere und konfiguriere [Hytale-API](https://github.com/xXSchrandXx/de.xxschrandxx.hytale-api/blob/main/Setup.md) und alle erforderlichen Pakete und Plugins.
2. Installiere [Hytale-Linker](https://www.woltlab.com/pluginstore/file/TODO/) über den Storecode, über die Paketsuche oder als Dateiupload.
3. Konfiguriere im ACP unter `Konfiguration > Optionen > Hytale > Hytale-Linker` Hytale-Linker. (Es muss ein Server in der Auswahl aktiviert werden damit Hytale-Linker funktioniert.)
4. Merke dir die URL für `sendCode`.
5. Merke dir die URL für `updateNames`.
6. Merke dir die URL für `getLinked`.
7. Merke dir die URL für `getUnlinked`.
8. Installiere [WSC-Linker](https://github.com/xXSchrandXx/WSC-Hytale-Linker/blob/main/Setup.md) auf deinem Hytale-Server.
