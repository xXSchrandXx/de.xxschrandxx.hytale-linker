# General
## Description
This plugin is an interface between other plugins and the Hytale-API.
Hytale Linker lets users link their Hytale UUID (s) to their WSC account.
It does not matter whether `online-mode` is activated or deactivated. The user will be sent a confirmation code on the Hytale server with which they can unlock themselves in the WSC.
## Requirements
1. [Hytale-API](#links) installed on WoltLab.
2. [WSC-Hytale-Bridge](#links) installed on your Bukkit- / Spigot- / BungeeCord-Server.
